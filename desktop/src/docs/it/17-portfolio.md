# Gestione Portfolio

Il modulo portfolio di LibrePM permette di raggruppare i progetti in programmi e monitorarli a un livello superiore.

## Programmi

Un programma è un insieme di progetti correlati gestiti insieme per raggiungere obiettivi strategici.

- Crea programmi con nome, descrizione e responsabile
- Monitora lo stato del programma: Attivo, Sospeso, Completato, Archiviato

## Appartenenza ai Progetti

Aggiungi progetti a un programma per visualizzarli insieme:
- Ogni progetto può appartenere a più programmi
- Visualizza salute, stato e date chiave del progetto a colpo d'occhio

## Milestone di Programma

Definisci milestone a livello di programma che attraversano più progetti:
- Imposta date obiettivo
- Collega le milestone a progetti specifici
- Monitora lo stato: In attesa, In corso, Completata, Scaduta

## Dashboard di Riepilogo

La panoramica del programma mostra:
- Numero totale di progetti
- Avanzamento delle milestone
- Schede riepilogo progetto con indicatori di salute

## Per Iniziare

1. Vai alla pagina **Portfolio**
2. Crea un nuovo programma
3. Aggiungi progetti esistenti al programma
4. Definisci le milestone del programma
5. Monitora l'avanzamento dalla panoramica
