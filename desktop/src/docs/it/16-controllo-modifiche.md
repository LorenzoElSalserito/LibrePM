# Controllo Modifiche e Branch

LibrePM fornisce la gestione formale delle modifiche tramite richieste di modifica, branch di progetto e registro delle decisioni.

## Richieste di Modifica

Gestisci le modifiche al progetto in modo formale:
- **Titolo e Descrizione**: Cosa deve cambiare
- **Motivazione**: Perché la modifica è necessaria
- **Ambito**: Cronoprogramma, Budget, Ambito, Risorse o Tutto
- **Priorità**: Bassa, Media, Alta, Critica
- **Impatto Previsto**: Quali effetti avrà la modifica

Le richieste di modifica attraversano gli stati: Bozza, Inviata, In Revisione, Approvata, Respinta, Implementata.

## Branch di Progetto

Crea branch di scenario per esplorare alternative senza influenzare il progetto principale:
- **SCENARIO** - Analisi what-if
- **RIPREVISIONE** - Proiezioni aggiornate
- **CANDIDATURA** - Varianti di gara/offerta
- **PIVOT** - Cambio di direzione strategica
- **SPINOFF** - Derivazione di nuovo progetto
- **EVOLUTIVO** - Miglioramenti incrementali

I branch possono essere collegati a richieste di modifica e monitorati tramite gli stati: Attivo, Approvato, Respinto, Unito o Archiviato.

## Registro delle Decisioni

Mantieni un registro permanente delle decisioni di progetto:
- Cosa è stato deciso
- La motivazione alla base della decisione
- Chi ha preso la decisione
- Riepilogo dell'impatto
- Collegamenti a branch o richieste di modifica correlate

Questo crea una traccia di audit del perché il progetto si è evoluto in quel modo.
