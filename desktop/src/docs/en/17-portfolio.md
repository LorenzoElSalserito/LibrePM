# Portfolio Management

LibrePM's portfolio module lets you group projects into programmes and monitor them at a higher level.

## Programmes

A programme is a collection of related projects managed together to achieve strategic objectives.

- Create programmes with name, description, and owner
- Track programme status: Active, On Hold, Completed, Archived

## Project Membership

Add projects to a programme to view them together:
- Each project can belong to multiple programmes
- View project health, status, and key dates at a glance

## Programme Milestones

Define programme-level milestones that span across projects:
- Set target dates
- Link milestones to specific projects
- Track status: Pending, In Progress, Completed, Overdue

## Overview Dashboard

The programme overview shows:
- Total number of projects
- Milestone completion progress
- Project summary cards with health indicators

## Getting Started

1. Navigate to the **Portfolio** page
2. Create a new programme
3. Add existing projects to the programme
4. Define programme milestones
5. Monitor progress from the overview
