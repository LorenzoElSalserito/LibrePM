# Chapter 10: Templates & Gallery 🎨

Don't want to build every project from scratch? The **Gallery** page offers a collection of pre-configured project templates that give you a head start.

---

## 🖼️ The Template Gallery

Go to the **"Gallery"** page in the sidebar. You'll see a grid of template cards.

### Template Card
Each template card shows:
*   **Icon**: A visual identifier for the template.
*   **System Badge**: If the template is built into LibrePM, a "System" badge appears.
*   **Name**: The template name (e.g., "Project Timeline", "Event Marketing Timeline").
*   **Description**: A brief explanation of what the template is for (3-line preview).
*   **Task Count**: How many tasks are included (list-task icon + number).
*   **Milestone Count**: How many milestones are included (flag icon + number).
*   **"Preview & Use"** button: Opens the template preview.

### Built-in Templates
LibrePM comes with 4 system templates:
*   **Project Timeline**: A general-purpose project timeline.
*   **Project Tracking**: For tracking project execution.
*   **Gantt Project**: A template designed for Gantt-based planning.
*   **Event Marketing Timeline**: For planning marketing events.

---

## 🚀 Using a Template

Click **"Preview & Use"** on any template card. A modal opens with:

### Left Column: Template Details
*   **Template name** and **description**.
*   **Included Features**: A bulleted list of what the template includes (tasks, phases, milestones, etc.).

### Right Column: Your New Project
*   **Project Name** (required): Choose a name for your new project.
*   **Description**: Add an optional description.

### Creating the Project
1. Fill in the project name.
2. Optionally add a description.
3. Click **"Create Project"** (rocket icon).
4. LibrePM creates a fully configured project with all the template's tasks, phases, and structure pre-populated.
5. A spinner and "Creating..." text show during the creation process.

After creation, you're redirected to start working on your new project!
