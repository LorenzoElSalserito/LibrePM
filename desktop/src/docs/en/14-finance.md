# Project Finance

LibrePM includes a complete project finance module for managing budgets, funding sources, and cost tracking.

## Budget Management

- Create multiple budget versions per project
- Define budget lines by category (Personnel, Suppliers, Software, Travel, Events, Equipment, Overhead, Misc)
- Track planned, committed, actual, and forecast amounts per line
- Approve budgets to lock them as baselines

## Funding Sources

- Register funding sources with type (Internal, Grant, Sponsor, Donation, Self-funded)
- Track funding amounts, currency, and restrictions
- Monitor funding coverage against budget

## Cost Tracking

- Record actual costs against budget lines
- Track cost dates and descriptions
- View burn rate and variance analytics

## Analytics

The analytics tab provides:
- **Total Planned vs Actual**: overall budget health
- **Variance**: how far off-budget you are
- **Burn Rate**: visual progress bar showing actual spend vs plan
- **Funding Coverage**: how much of the budget is covered by funding
- **Category Breakdown**: actual costs by budget category

## Getting Started

1. Navigate to the **Finance** page
2. Create a new budget
3. Add budget lines for your cost categories
4. Add funding sources
5. Record actual costs as the project progresses
